SportUp

Inspired by the popular webapp 'Meetup', SportUp is built with Rails, JavaScript and React. This app uses Flux architecture, user authentication with BCrypt, and a PostgreSQL database. SportUp is a one-page-app, which takes advantage of a RESTful JSON API and AJAX requests to rerender the frontend without making a full HTTP request. This allows the app to operate extremely fast.
